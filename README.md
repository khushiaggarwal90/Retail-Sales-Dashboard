                                                📊 Retail Sales Dashboard  



 📌 Project Overview

This project is a Retail Sales Dashboard created using  Google Sheets.  
It provides meaningful insights into sales performance using Pivot Tables, Charts, and Interactive Slicers.

The dashboard helps in analyzing sales data based on different categories such as product, category, sub-category, and date.

---

 📂 Dataset Details

The dataset used in this project contains the following columns:
- Order ID  
- Date  
- Category  
- Sub Category  
- Product 
- Quantity  
- Unit Price  
- City
-Region
-Payment Mode
-Total Sales

---

 ⚙️ Features

- 📊 Interactive Dashboard  
- 📌 Pivot Tables for data summarization  
- 📈 Charts (Bar, Line, Pie) for visualization  
- 🎛️ Slicers for filtering data dynamically  
- 🎨 Clean and visually appealing design  

---

 📊 Dashboard Insights

The dashboard provides insights such as:
- Payment Mode Analysis
- Category-wise sales distribution  
- Monthly/Date-wise sales trends  

---

 🚀 How to Use

1. Open the Google Sheets link  
2. Use slicers to filter data (Category, Payment Mode, etc.)  
3. Analyze charts and insights
